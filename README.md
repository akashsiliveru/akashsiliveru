## Projects

### Credit card fraud detection
End-to-end ML pipeline to detect fraudulent transactions from a heavily imbalanced dataset (~0.17% fraud rate). Compared Logistic Regression, Random Forest, and XGBoost; optimised for recall and F1 to minimise false negatives.

**Key contributions**
- Handled class imbalance with SMOTE and undersampling strategies
- Feature scaling + correlation-based feature selection
- Model evaluation using precision, recall, F1, and ROC-AUC

**Technologies:** Python · Pandas · NumPy · Scikit-learn · Seaborn · SMOTE

🔗 [View repository](https://github.com/akashsiliveru/credit-card-fraud-detection)

### Arogya Plan — diet recommendation system
ML-powered web app that generates personalised meal plans (breakfast, lunch, dinner, snacks) based on a user's health profile. Takes BMI, activity level, dietary preference, blood sugar, cholesterol, and fitness goal as inputs and finds the closest matching nutrition plan using KNN.

**Key contributions**
- Built KNN-based recommendation engine with StandardScaler preprocessing
- Handled multi-goal support: weight loss, gain, maintenance, and healthy lifestyle
- Deployed as an interactive Streamlit app with a live public demo

**Technologies:** Python · Scikit-learn · Pandas · NumPy · Streamlit · Pickle

🔗 [View repository](https://github.com/akashsiliveru/diet-recommendation-system) &nbsp;·&nbsp; 🚀 [Live demo](https://diet-recommendation-system-2drzcfznrwqxxtr2ou6vny.streamlit.app/)


### Nyay AI — multilingual legal assistant
AI-powered assistant that helps Indian citizens understand complex legal language. Users describe a situation in plain language and the system explains the relevant laws, maps them to specific IPC/BNS sections, and responds in multiple Indian languages to bridge the awareness gap.

**Key contributions**
- Built NLP pipeline to parse legal queries and match relevant sections from a structured law dataset
- Implemented multilingual response support to address language barriers
- Designed for accessibility — plain language input, structured legal output

**Technologies:** Python · NLP · LLM API · PDF parsing

🔗 [View repository](https://github.com/akashsiliveru/nyay-AI)
